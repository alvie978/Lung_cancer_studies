This project investigates various factors that may contribute to the development of lung cancer, with the broader goal of informing prevention strategies.

## 📖 Project Overview
Using statistical analysis and data visualization, this study explores correlations between lifestyle, demographic, and environmental factors and the incidence or risk of lung cancer.

## 📁 Files Included
*   **`Lung_cancer_Analysis_Project.ipynb`**: A Jupyter Notebook (created with Google Colab) containing the full data analysis, including data cleaning, exploratory data analysis (EDA), and statistical modeling.
*   **`Lung Cancer Dashboard.pbix`**: The Power BI report file containing the interactive dashboard and all its visualizations.
*   **`Lung Cancer Dashboard.pdf`**: A PDF export of the Power BI dashboard, providing a static view of the key charts and findings for easy sharing.
*   **`Lung Cancer.csv`**: The dataset used for this project, containing records on patient attributes, risk factors, and diagnoses.

## 🛠️ Tools Used
*   **Python** (Pandas, Scikit-learn, Matplotlib/Seaborn) for data processing and analysis.
*   **Google Colab** for the analysis notebook.
*   **Microsoft Power BI** for creating the interactive dashboard.

## 🚀 How to Use This Project
1.  **Understand the Data**: Look at `Lung Cancer.csv` to see the variables used in the analysis.
2.  **Follow the Analysis**: Run `Lung_cancer_Analysis_Project.ipynb` in Google Colab or Jupyter to see the code and findings step-by-step.
3.  **Explore the Dashboard**:
    *   For an interactive experience, download `Lung Cancer Dashboard.pbix` and open it with Power BI Desktop.
    *   For a quick overview, review the static charts in `Lung Cancer Dashboard.pdf`.
